# Creeer een Lijst van gewenste personen voor IP-Gebaseerde Toegang API {#create-an-allowlist-for-ip-based-api-access}

Soms wilt u API-toegang alleen verlenen aan een specifiek IP-adres of een reeks adressen. Hiervoor moet u eerst beperkingen inschakelen en vervolgens de IP-adressen opgeven die de API&#39;s mogen gebruiken.

>[!NOTE]
>
>**Beheerdersmachtigingen vereist**

1. Ga naar **Admin** en klik de Diensten **van het** Web.

   ![](assets/image2016-2-25-9-3a12-3a48.png)

1. Klik in het gedeelte IP-beperkingen op **Bewerken **of klik** op IP-beperkingen bewerken **linksboven.

   ![](assets/image2016-2-25-9-3a15-3a30.png)

1. Controleer de **Enable IP doos van Beperkingen** en ga de IP adressen in u lijst van gewenste personen wilt.

   ![](assets/image2016-2-25-9-3a18-3a28.png)

   >[!NOTE]
   >
   >U kunt één enkel IP adres of een waaier van hen ingaan, of een vervanging gebruiken.

1. Klik **toevoegen** om extra gebieden te openen om meer IP adressen in te gaan.

   ![](assets/image2016-2-25-9-3a20-3a47.png)

1. Klik op **Opslaan**.

   ![](assets/image2016-2-25-9-3a28-3a21.png)
