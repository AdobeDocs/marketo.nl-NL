# Overzicht van Web Personalization {#web-personalization-overview}

## Inleiding {#introduction}

Voor nieuwe Marketo-klanten is [!UICONTROL Real-Time Personalization] verdeeld in vier apps, die u kunt openen met de volgende twee blokken: [!UICONTROL Web Personalization] en [!UICONTROL ContentAI] .

![](assets/pasted-image-at-2016-03-23-02-45-pm.png) ![](assets/mlm-homepage-content-ai-281-29.png)

>[!NOTE]
>
>[!UICONTROL Web Personalization] en [!UICONTROL ContentAI] zijn beschikbaar in alle door Marketo ondersteunde talen (Engels, Frans, Duits, Japans, Portugees, Spaans).

Bestaande klanten blijven toegang krijgen tot de apps via de Real-Time Personalization-tegel tot de vernieuwing, waarna ze de bovenstaande aanmelding gebruiken.

![](assets/image2016-2-9-8-3a52-3a32.png)

Wanneer u op de tegel [!UICONTROL Web Personalization] klikt, wordt de pagina Dashboard weergegeven. Klik op de tegel [!UICONTROL ContentAI] en ga rechtstreeks naar de pagina Aanbevelingen voor inhoud.

De vier toepassingen zijn:

* [!UICONTROL Web Personalization]
* [!UICONTROL Account-Based Web Marketing]
* [!UICONTROL Website Retargeting]
* [!UICONTROL ContentAI]

De tegel **[!UICONTROL Web Personalization]** is het toegangspunt voor het maken van gepersonaliseerde webcampagnes. Op account-gebaseerde webmarketing en het opnieuw samenstellen van het web. U kunt inhoud toevoegen op de Content Analytics-pagina.

**[!UICONTROL Account-Based Web Marketing]** werkt met Web Personalization en beschikt over benoemde accounts. De pagina Benoemde accounts biedt grafische informatie over bestaande klanten, totale contracten en trends in de organisatie, en een lijst met accounts. U kunt daar ook nieuwe benoemde accounts maken.

**[!UICONTROL Website Retargeting]** geeft u de macht om aan gesegmenteerd publiek opnieuw in de handel te brengen. U kunt segmenten instellen, maar u kunt benoemde accounts niet gebruiken in het filter.

**[!UICONTROL ContentAI]** is de plaats om inhoudaanbevelingen te geven. U kunt inhoud eenvoudig toevoegen aan de pagina Aanbevelingen voor inhoud.

>[!NOTE]
>
>Als u [!UICONTROL Account-Based Web Marketing] of [!UICONTROL Website Retargeting] wilt openen, moet u op de [!UICONTROL Web Personalization] -tegel klikken, zelfs als u de [!UICONTROL Web Personalization] -app niet hebt.

## Organisaties {#organizations}

Op het tabblad Organisaties worden alle gegevens (naam, locatie, activiteit en tijdstempel) weergegeven van organisaties die uw website gedurende een bepaalde periode hebben bezocht. De tabel kan worden gesorteerd en geordend op tijd, locatie, domein en via een vrije tekstzoekopdracht.

![](assets/image2014-11-10-19-3a23-3a18.png)

>[!TIP]
>
>&quot;Meest actief&quot; versus &quot;Nieuwst&quot; - De tabel wordt georganiseerd door geïdentificeerde organisaties en vervolgens door internetserviceproviders (aangeduid met een ISP-pictogram). Het kan worden gesorteerd op:
>
>* Meest actief: de meest actieve organisaties in de tabel, op basis van het aantal paginaweergaven
>* Laatste: de meest recente organisaties in de tabel (standaard)
>* Voor waardevolle inzichten, filter volgens het meest actieve

**Organisaties van het Aandeel**

Wilt u de gegevens van uw websitebezoeker delen met een collega? Klik op een organisatie en klik vervolgens op de koppeling Delen in het rechterdeelvenster.

Hiermee activeert u de [!UICONTROL Share this Organization] -overlay, waar u een e-mail kunt verzenden met daarin de naam en de details van de organisatie die de website heeft bezocht.

![](assets/image2014-11-10-19-3a25-3a42.png)
