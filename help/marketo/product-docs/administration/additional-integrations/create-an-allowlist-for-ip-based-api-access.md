# Creeer een Lijst van gewenste personen voor op IP-Gebaseerde Toegang API {#create-an-allowlist-for-ip-based-api-access}

Soms wilt u API-toegang alleen verlenen aan een specifiek IP-adres of een reeks adressen. Hiervoor moet u eerst beperkingen inschakelen en vervolgens de IP-adressen opgeven die de API&#39;s mogen gebruiken.

>[!NOTE]
>
>**Beheerdersmachtigingen vereist**

1. Ga naar **Admin** en klik **Webservices**.

   ![](assets/image2016-2-25-9-3a12-3a48.png)

1. Klik in het gebied IP-beperkingen op **Bewerken,** of klik op **IP-beperkingen bewerken** linksboven.

   ![](assets/image2016-2-25-9-3a15-3a30.png)

1. Controleer **laat IP Beperkingen** vakje toe en ga de IP adressen in u lijst van gewenste personen wilt.

   ![](assets/image2016-2-25-9-3a18-3a28.png)

   ![](assets/image2016-2-25-9-3a20-3a47.png)

1. Klik **Opslaan**.
