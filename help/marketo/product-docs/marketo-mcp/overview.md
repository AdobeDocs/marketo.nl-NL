# Document MCP-overzicht {#overview}

TEKST VAN ATHER:

De Public MCP Server voor Marketo Engage is een cloudtoegankelijke dienst die de volledige macht van de Marketo Engage openbare REST APIs als gestructureerde, ontdekkbare hulpmiddelen voor AI agenten blootstelt. Op basis van het ModelContext Protocol (MCP) wordt de Marketo-mogelijkheden voor marketingautomatisering getransformeerd in een gestandaardiseerde, schemagebaseerde interface die AI-systemen veilig en intelligent kunnen gebruiken.

In plaats van het manueel integreren van Marketo eindpunten in elke toepassing, handelt de Server MCP als enige, inheemse gateway AI die agenten toestaat MCP compatibele om beschikbare verrichtingen te ontdekken, vereiste parameters te begrijpen, en acties door een verenigbaar protocol uit te voeren.

Er zijn een groot aantal beschikbare hulpmiddelen die bestaan uit zowel openbare REST API&#39;s als interne API&#39;s, die de kernfuncties van Marketo uitvoeren zoals in de gebruikersinterface maar via API-aanroepen. Naarmate gereedschappen worden ontwikkeld, worden er meer gereedschappen toegevoegd.

De server MCP wordt aangeboden als zelfbediening en kan opstelling door het kader van Adobe IO door een eindpunt te bouwen en de verbinding voor authentiek te verklaren. Ga voor meer informatie en instellingsgegevens naar de Adobe Developer-site op (klik hier voor een link).

Campagneprogramma&#39;s die als SOAP API&#39;s worden weergegeven, zijn geconverteerd naar REST API&#39;s en toegevoegd.

Wanneer gevormd, om een lijst van beschikbare Hulpmiddelen te bekijken, volg deze instructies hier (hyperlink).
