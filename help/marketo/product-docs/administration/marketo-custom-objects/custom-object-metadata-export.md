# Metagegevens van aangepast object exporteren {#custom-object-metadata-export}

Als u onze SOAP API of Munchkin API gebruikt, kan het Schema van de Meta-gegevens van de Objecten van de Douane worden uitgevoerd. Zo krijg je het.

1. In Mijn Marketo, klik **Admin**.

   ![](assets/custom-object-metadata-export-1.png)

1. Klik **Marketo Custom Objects**.

   ![](assets/custom-object-metadata-export-2.png)

1. Selecteer het markeerteken naar aangepast object dat u wilt exporteren.

   ![](assets/custom-object-metadata-export-3.png)

1. Klik op de vervolgkeuzelijst **Aangepaste objecthandelingen** en selecteer **Object exporteren**.

   ![](assets/custom-object-metadata-export-4.png)
