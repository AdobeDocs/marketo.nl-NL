# Document MCP-overzicht {#overview}

TEKST VAN ATHER:

De openbare server MCP van Marketo Engage is een op wolk-gebaseerde dienst die Marketo REST APIs voor AI agenten als gestructureerde hulpmiddelen toegankelijk maakt. Voortgebouwd op het ModelProtocol van de Context (MCP), verstrekt het een gestandaardiseerde, schema-gedreven interface die AI systemen toestaat om veilig en efficiënt met Marketo in wisselwerking te staan.

In plaats van het manueel integreren van Marketo eindpunten in elke toepassing, verstrekt de Server MCP één enkele, AI-inheemse gateway die agenten MCP-Compatibele toestaat om beschikbare verrichtingen te ontdekken, vereiste parameters te begrijpen, en acties door een verenigbaar protocol uit te voeren.

Er zijn een groot aantal beschikbare hulpmiddelen die bestaan uit zowel openbare REST API&#39;s als interne API&#39;s, die de kernfuncties van Marketo uitvoeren zoals in de gebruikersinterface maar via API-aanroepen. Naarmate gereedschappen worden ontwikkeld, worden er meer gereedschappen toegevoegd. &lt;— NODIGE VERDUIDELIJKING

De server MCP wordt aangeboden als zelfbediening en kan opstelling door het kader van Adobe IO door een eindpunt te bouwen en de verbinding voor authentiek te verklaren. Ga voor meer informatie en instellingsgegevens naar de Adobe Developer-site (voeg hier de koppeling toe). DEZE? https://developer.adobe.com/marketo-apis/

Campagneprogramma&#39;s die als SOAP API&#39;s worden weergegeven, zijn geconverteerd naar REST API&#39;s en toegevoegd.

Wanneer gevormd, om een lijst van beschikbare Hulpmiddelen te bekijken, volg deze instructies hier (hyperlink).

NOTA: De hulpmiddelen die niet beschikbaar zijn zijn &quot;Deactivate Campagne&quot;en &quot;schrappen Statische Lijst&quot; (TBD).
